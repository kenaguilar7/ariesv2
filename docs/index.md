# Aries Contabilidad

Welcome to the Aries Contabilidad documentation. This accounting software is designed to help businesses manage their financial records efficiently.

## Features

- Chart of Accounts Management
- Journal Entries
- Financial Reports
- Balance Sheet
- Income Statement
- Trial Balance
- Multi-company Support
